### **What is in a "Fullz"?**
In the underground economy, the term **"Fullz"** refers to a "full" set of information—a complete dossier that gives a criminal everything they need to commit high-level identity theft or synthetic identity fraud. 

While a single credit card number is just a "card," a "Fullz" record is a person’s entire digital and legal shadow. A standard 2026 Fullz package includes:

1.  **Core Identity:**
    * **Full Legal Name**
    * **Social Security Number (SSN):** The "anchor" of the record.
    * **Date of Birth (DOB):** Used to bypass security questions.
2.  **Contact & Location:**
    * **Current Home Address:** (Often including previous addresses to pass "knowledge-based authentication" checks).
    * **Phone Number:** Used for SIM-swapping or intercepting 2FA codes.
    * **Email Address & Password:** Often the password for the email is included if it was harvested in the same breach.
3.  **Financial Foundation:**
    * **Bank Account & Routing Numbers.**
    * **Credit/Debit Card Details:** Number, Expiry, and CVV/CVC.
4.  **Verification Metadata:**
    * **Driver’s License Number:** Sometimes includes a high-res scan of the ID.
    * **Mother’s Maiden Name:** A common "secret" security question.
    * **IP Address/User Agent:** The specific digital signature of the victim's computer (hackers use this to "mimic" the victim’s device so the bank doesn't flag the login as "new").

**Why "Fullz" are dangerous:** Unlike a stolen credit card, which you can replace in 5 minutes, a "Fullz" set allows a criminal to take out a mortgage, apply for a $50,000 car loan, or file a fraudulent tax return in your name. They are the most versatile "tool" in a cybercriminal's kit.
